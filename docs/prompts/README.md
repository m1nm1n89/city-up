## 使い方

新しいフェーズのプロンプトをClaude Codeに渡すとき、冒頭に以下を含める:

> このフェーズに入る前に、`docs/prompts/` 配下の過去プロンプトを確認してください。
> 過去のフェーズで決めたルール・命名規則・ファイル構造を踏襲してください。

## ⚠️ フェーズ完了時の必須作業

**各フェーズのプロンプトに「このREADME.mdを更新せよ」という指示を含めている**。
Claude Codeがフェーズ完了報告を出す前に、この表の「状態」列を🚧→✅に更新する。

もし更新が漏れた場合は、ユーザー(あなた)が手動で更新する。

## 完了報告のフロー

1. Claude Code がフェーズの実装を完了
2. Claude Code がこの `README.md` の進捗表を更新(🚧→✅)
3. Claude Code が完了報告を出す
4. ユーザーが次フェーズのプロンプトをClaude(Web版)に依頼
